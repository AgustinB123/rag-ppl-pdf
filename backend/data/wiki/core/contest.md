# Contest
Contest es un marco de prueba unitario minimalista, multiplataforma para .NET.

Dentro del core v6,  utilizamos la herramienta para ejecutar:
- Tests unitarios de interprete.
- Tests de integracion - Interprete, Operaciones, etc.

Podrá descargarse los fuentes y obtener mas información desde 
https://github.com/amiralles/contest
