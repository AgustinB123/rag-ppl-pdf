A continuación se describen las herramientas y prerrequisitos para incorporar desarrolladores al equipo core.


#### Lista de herramientas para instalar en su PC
1.	Visual Studio 2015
2.	ReSharper (opcional)
3.	SQL Management Studio
4.	SQL Developer
5.	Git
6.	BeyondCompare (O Araxis Merge, o Vimdiff, o algo por el estilo)
7.	Skype

#### Github
Validar que el programador tenga una cuenta en github. Esto es fundamental para que pueda acceder a los distintos repositorios que utiliza el equipo core. Actualmente estos repositorios son privados, por lo tanto es necesario que un admin lo agregue como colaborador para esos repositorios.

#### Basecamp
Esta herramienta se utiliza para gestionar la comunicación entre los distintos equipos internos de FPA, proponer ideas, nuevos features para la aplicación, seguimiento de tareas, emails y demás. Para poder participar de este workflow el programador necesita una cuenta.

**Basecamp ya no se utiliza más, unicamente Trello**

#### Trello
Esta herramienta es una especie de kanban borad que actualmente solo utiliza el equipo del core para hacer el seguimiento de las distintas tareas que realizan. Al igual que con Basecamp, el recurso tiene que contar una cuenta para poder participar.


_* Nota: Si bien es cierto que un programador "core" podría instalar todo por su cuenta, es mucho más prolijo que al momento de la incorporación cuente con todas las herramientas instaladas y configuradas correctamente._
