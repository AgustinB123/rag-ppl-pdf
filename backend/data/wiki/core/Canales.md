# Canales de Comunicacion

Tenemos en FPA definidas distintas vias de comunicacion para las consultas, reportes de errores y pedidos de desarrollos.
  
## Pedidos de cotizacion o modificaciones de core *(por error o nueva funcionalidad)*

Si encontramos un error en la aplicacion o queremos pedir una funcionalidad esto tiene un circuito de requerimientos y para eso usamos el trello teniendo en cuenta las siguientes pautas : [Pedidos de modificacion de core: Circuito ](/core/Circuito-de-requerimientos) .

## Consultas o Comunicados
 Para realizar consultas o tratar algun tema especifico del cual todos tengamos que estar al tanto usamos canales de chat de google a continuacion contamos que canales existen y sus propositos.
 
**- Soporte PPL Core V6**

En esta sala se vuelcan las dudas y los problemas de uso de PPL para ser atendidos por el equipo de Core, sin embargo cualquier integrante puede aportar si sabe la solucion al problema que esta teniendo su par.
Una consulta podria derivar en un issue de trello en caso de requerir tiempos de core.
