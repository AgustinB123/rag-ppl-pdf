# Introducción

Listado de funcionalidades nuevas desarrolladas (y en desarrollo) que tiene V6.
Estas funcionalidades no existen en V3.

Contienen:
- Features nuevos del lenguaje PPL.
- PPLStudio: herramientas de desarrollo.
- Portfolio: features orientados al usuario.
- Ecosistema: Integraciones con aplicaciones y servicios

# Listado


## Ecosistema

- [App Server](/appserver/App-Server): Acceso a datos desde servidor remoto.
- [Hasp](/instalacion/Hasp): Servicio de licencias.
- [AC32](/ac32/inicio): Nueva versión del distribuidor de scripts PPL.
- [FPA Console](/instalacion/fpa-console): Nueva versión de ejecución de PPL desatendida.
- [Launcher](/core/launcher) Optimizacion de instalación centralizada.
- [FPA HUB](/v6/fpa-hub): Servicio de notificaciones real-time. (En desarrollo)
- [FPA Sync](/core/fpa-sync): Scheduler de ejecuciones PPL remotas. (En desarrollo)
