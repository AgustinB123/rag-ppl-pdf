# Shortcuts en PPLStudio

En el editor de script del PPLStudio, además de los shortcuts (atajos) especificados en el menú Edit de la barra principal, existen otros shortcuts para agilizar el codeo en el script PPL.
Éstos shortcuts son los siguientes:

| Shortcut         | Descripción|
|------------------|------------|
|Ctrl + F|Abre el dialogo para buscar caracteres|
|Ctrl + H|Abre el dialogo para reemplazar caracteres|
|F3|Busca la siguiente aparición de la clave ingresada en el dialogo del buscador|
|Tab / Shift + Tab|Indenta/Desindenta la porción de código seleccionada|
|Ctrl + Inicio|Va al principio del script|
|Ctrl + Fin|Va al final del script|
|Ctrl + U|Convierte el texto seleccionado en mayúscula|
|Shift + Ctrl + U|Convierte el texto seleccionado en minúscula|
|Alt + Shift + (Up, Down, Right, Left)|Habilita la selección de código por columna|
|Alt + Up, Alt + Down|Mueve la línea para arriba o para abajo|
|Ctrl + M|Comienza la grabación del macro. También la finaliza|
|Ctrl + E|Ejecuta la macro grabada|
|Ctrl + (Up, Down)|Scrollea el script hacia arriba o hacia abajo|
